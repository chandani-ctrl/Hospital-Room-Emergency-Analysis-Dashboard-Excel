🏥 Hospital Emergency Room Dashboard (Excel)
📌 Project Overview

This project is an interactive Hospital Emergency Room Dashboard created using Microsoft Excel.
It provides a clear monthly and yearly overview of patient flow, wait times, admission status, satisfaction levels, and department referrals to help hospital management make data-driven decisions.

The dashboard is designed to be simple, visual, and interview-ready, suitable for showcasing data analysis and dashboarding skills.

🎯 Objectives

Analyze Emergency Room performance

Track patient volume and trends

Monitor average wait time

Measure patient satisfaction

Understand admission status

Analyze gender, age group, and department referrals
